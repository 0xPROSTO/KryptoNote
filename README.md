# ZeroXX-KryptoNote

Система управления знаниями на базе интерактивного графа с упором на безопасность данных.

### Основные возможности
* **Бесконечный холст:** Свободное размещение узлов (nodes) и управление масштабом.
* **Типы узлов:** Поддержка текста, изображений и видео.
* **Связи:** Создание визуальных линий между узлами для структурирования информации.
* **Поиск:** Быстрая навигация по заголовкам и контенту заметок.

### Безопасность и Архитектура
* **Шифрование:** AES-256-GCM (с проверкой целостности данных).
* **Хеширование:** Argon2id для генерации ключа из пароля.
* **Diskless Streaming:** Дешифровка видео происходит напрямую в RAM. Программа не создает временных файлов (Temp) на диске.
* **Chunking:** Крупные файлы шифруются и хранятся блоками по 4 МБ, что позволяет работать с тяжелым контентом без перегрузки ОЗУ.
* **Хранилище:** SQLite в режиме WAL для надежной записи данных.

### Установка и запуск
Требуется Python 3.10+ и установленные зависимости из `requirements.txt`.

```bash
pip install -r requirements.txt
python main.py
```
